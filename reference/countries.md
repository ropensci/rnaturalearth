# world country polygons from Natural Earth

at 1:110m scale (small). Other data and resolutions are in the packages
rnaturalearthdata and rnaturalearthhires.

## Usage

``` r
countries110
```

## Format

A `sf` object.

An object of class `sf` (inherits from `data.frame`) with 177 rows and
169 columns.

## Source

<https://naciscdn.org/naturalearth/10m/cultural/ne_10m_admin_0_countries.zip>

## Slots

- `data`:

  A data frame with country attributes.
